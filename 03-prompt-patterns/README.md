# Prompt Engineering Patterns

## Co je nového?

**Chain of Thought (CoT)** - Model "přemýšlí" krok za krokem
**ReAct Pattern** - Reasoning + Acting v cyklu
**Few-shot s dynamickými příklady** - Kontextově relevantní příklady
**Prompt Caching** - 90% úspora na opakovaném contextu

## Ukázky

1. **chain-of-thought.js** - Krok za krokem reasoning
2. **react-pattern.js** - Reasoning + Acting loop
3. **few-shot-dynamic.js** - Dynamické příklady podle kontextu
4. **prompt-templates.js** - Reusable prompt šablony
